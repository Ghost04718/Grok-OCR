Welcome to Grok OCR!

Thank you for installing the Grok OCR Extension. This tool helps you extract text from any image on the web using grok-2-vision, a powerful AI model for visual recognition tasks.

1
Set up your API key
Enter your Grok API key in the extension's settings page. This is required to use the OCR functionality.
2
Right-click on any image
Find an image containing text on any webpage, right-click on it, and select "Perform OCR on this image".
3
View the extracted text
The text from the image will be extracted and displayed in the extension popup. You can then copy, download, or otherwise use the text.

Tips & Tricks

Here are some tips to help you get the most out of the extension:

  • Use preset prompts for specific types of text extraction (preserving layout, converting to markdown, etc.)
  • Try the context menu's "OCR with preset..." option for quick access to different extraction styles
  • For better results with complex images, consider using specific prompts that describe the type of content
  • Enable dark mode for comfortable nighttime use
  • Use keyboard shortcuts for faster workflow
